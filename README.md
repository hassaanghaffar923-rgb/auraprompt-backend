# 🚀 AuraPrompt — AI Video Prompt Studio

AuraPrompt is a 100% free, structured prompt engineering studio designed for AI video creators. Craft high-quality, cinematic prompts optimized for advanced text-to-video models like Runway Gen-3, Luma Dream Machine, and Sora.

🌐 **Live Website:** [https://netlify.app](https://netlify.app)  
🚀 **Product Hunt Launch:** [Join our Waitlist / Support Us!](#)

---

## 🔥 Key Features

- **Granular Camera Controls:** Select exact camera movements (Dolly, Pan, Zoom) and angles.
- **Cinematic Lighting & Styles:** Apply lighting presets (Golden Hour, Neon, Cyberpunk) with one click.
- **Multi-Variation Generation:** Generate 3 distinct prompt variations simultaneously using Groq API.
- **Smart Rate Limiting:** Built-in 5 prompts per day per user protection.

## 🛠️ Tech Stack

- **Frontend:** React / Next.js (Hosted on Netlify)
- **Backend:** Node.js / Express (`server.js`)
- **AI Engine:** Groq API (Llama 3 / Mixtral)
